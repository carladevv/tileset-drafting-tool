# Tileset Planner --- Technology Stack

## Core Stack

-   React
-   Vite
-   TypeScript

## State Management

-   Zustand

## Rendering / Visualization

-   HTML5 Canvas (tile editor + semantic grid)
-   Babylon.js (future 3D preview stages)

## Styling

-   CSS Modules
-   Vanilla CSS variables for theme colors

## Testing

-   Vitest

## File Import / Export

-   FileSaver.js

## Image Handling

-   Native browser APIs
-   HTMLImageElement
-   Canvas for fallback rendering

## Data Persistence

-   localStorage (MVP)
-   JSON export/import

## Key Packages

-   react
-   zustand
-   vitest
-   file-saver
-   @babylonjs/core (future stages)
-   @babylonjs/loaders (future stages)

## Development Principles

-   minimal dependencies
-   deterministic data model
-   React functional components
-   TypeScript-first architecture
